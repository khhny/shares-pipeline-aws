## Overview
README generated by AI:
End-to-end batch data pipeline ingesting historical stock price data
for top 10 VOO tickers (AAPL, MSFT, NVDA, GOOGL...) using AWS-native tools.

## Architecture
yfinance → S3 (raw CSV) → Glue Crawler → Glue ETL → S3 (Parquet) → Athena

## Features engineered
- Daily return % (close vs open)
- 20-day simple moving average (SMA-20)
- Price range (high - low)
- High-volume day flag (>2x ticker average)

## AWS Services used
| Service | Purpose |
|---|---|
| Amazon S3 | Raw + processed data lake storage |
| AWS Glue Crawler | Schema inference + Data Catalog registration |
| AWS Glue ETL (Python Shell) | Transformation + feature engineering |
| Amazon Athena | Serverless SQL analytics on S3 |
| Amazon QuickSight | Dashboard + visualization |
| AWS IAM | Least-privilege role-based access |

## Key findings (from Athena queries)
- MU had the highest avg daily return at 10.8% over the period
- MU had the largest price range consistently for the most amount of days over the period
- MU had the highest high volume days of 76 days in the period of 1119 total days
- NVDA's 20-day sma shows an overall decrease across the entire period.

## How to run
1. `pip install awscli boto3 yfinance pandas pyarrow`
2. `aws configure` with your IAM credentials
3. `python ingest_stocks.py` — downloads and uploads to S3
4. Run Glue crawler → Run Glue ETL job in AWS Console
5. Query in Athena using provided SQL in `/queries/`
